About this site

1. Making the site

For a long time I liked to have a homepage/blog:

  1. With pages in Tufte style, so that I can include remarks and figures in the margin;
  2. Exercises that have hidden, but clickable, hints and sotutions;
  3. Written in org mode and emacs.

These links helped me starting.

  1. https://parsec.ro/
  2. https://github.com/ctanas/parsec-lisp
  3. https://github.com/alhassy/org-special-block-extras

Here is some material that I should study at a later time:

  1. https://ogbe.net/blog/emacs_org_static_site

To make this site, run on the prompt in the root directory of the site: 

emacs --batch -l build-site.el

To view it locally, run python -m http.server in tmp/public. Then open localhost:8000 in Firefox.

2. Adding the houses

Run the bash script. 

bash bin/make_thumbs.sh

Update houses.org accordingly.
